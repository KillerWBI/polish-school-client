### `/calendar` — Календарь 🟡

**Файл:** `src/pages/calendar/CalendarPage.jsx`

👤 FullCalendar daygrid. Загружает групповые + индивидуальные уроки при смене периода. Клик на событие → модалка с деталями, ссылка на урок если есть.

**Проблемы:**
- Кнопки навигации на английском (нет импорта ru locale)
- Ошибка загрузки не показывается пользователю

### `/groups` — Список групп 🟡

**Файл:** `src/pages/groups/GroupsPage.jsx`

👨‍🏫 Список карточек групп + кнопка «Создать группу» (модалка с расписанием).
🎓 Сверху — блок **«Приглашения в группы (N)»** (C3): входящие приглашения от учителей с кнопками «Принять»/«Отклонить»; после accept группа появляется в списке.

**Проблемы:**
- Форма создания не сбрасывается при закрытии после ошибки
- Нет валидации `pricePerLesson < 0` на фронте

---

### `/groups/:id` — Детали группы 🟡

**Файл:** `src/pages/groups/GroupDetailPage.jsx`

👨‍🏫 Три вкладки: **Студенты** (добавить реального / «+ Заглушка» без аккаунта / «+ Пригласить» по нику (C3) / «Перенести» заглушку на реального / убрать), **Уроки** (список + генерация + создание/просмотр), **Настройки** (редактировать/удалить).

**Приглашение (C3, `InviteModal`):** поиск учеников **по похожим** (нику/имени) → список; на каждом «Пригласить» (или «Добавить», если уже ваш → прямое членство); ученик подтверждает у себя в `/groups`. _(security H1: без email, iLike-поиск)_

**`chatLink`:** поле «Ссылка группового чата» в настройках; если задана — кнопка «💬 Чат группы» в шапке группы (обе роли) и в карточке урока (здесь и в `/calendar`).

**Проблемы:**
- Нет UI пагинации (загружается максимум 50 уроков без сигнала об этом)

### `/profile` — Профиль (редизайн) ✅ (2026-05-27, Sprint C)

**Файл:** `src/pages/profile/ProfilePage.jsx`

👤 Instagram-style профиль для обеих ролей. Реализовано.

**Структура страницы:**
- **Верхняя зона:** обложка-баннер (coverImage, upload, 3:1 ratio) + круглый аватар (120px, upload поверх баннера) + кнопка «Редактировать профиль»
- **Инфо-блок:** имя (крупно), роль-бейдж, email (read-only), прогресс-бар заполненности профиля (%)
- **Статы-плашки (3 шт):**
  - Учитель: «Студентов», «Уроков проведено», «Средняя оценка»
  - Студент: «Уроков пройдено», «Посещаемость %», «ДЗ выполнено %»
- **Табы:** «Профиль» | «Аналитика»
  - Таб «Профиль»: bio (textarea, 300 симв.), языки (теги: учитель — «Преподаю», студент — «Изучаю»), соцсети (TG/WhatsApp/LinkedIn иконки + поля ввода)
  - Таб «Аналитика»: графики Recharts (см. ниже)

**Аналитика учителя (Recharts):**
- BarChart — доход по месяцам (последние 6 мес)
- LineChart — кол-во уроков по неделям (последние 8 нед)
- AreaChart — рост студентов по месяцам

**Аналитика студента (Recharts):**
- AreaChart — посещаемость по месяцам
- BarChart — оценки по ДЗ (последние 10)
- LineChart — динамика среднего балла
