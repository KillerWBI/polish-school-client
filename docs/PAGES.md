# Frontend — Страницы и функционал

**Обновлено 2026-07-14.**

> **📦 Актуальный список страниц и их статусы** — в `PROGRESS.md` (таблица «Страницы» + блок «Локализация»). Ниже — детальные описания части экранов; при расхождении приоритет у `PROGRESS.md`.
> **Локализация:** все описанные страницы учителя и общие (dashboard/quiz/calendar/settings/profile) переведены на **ru/en/pl/uk** через `useTranslation`; тексты — в `src/i18n/locales/<lang>/{landing,app,teacher}.json`. Страницы ученика (vocab/my-lessons/notes/progress/materials/topics/support) — перевод в очереди (`student.json`).
> **Актуальные маршруты входа:** `/login`, `/register`, `/register-student` (страница `AuthPage`, не модалка), `/for-students` (лендинг ученика). Старые упоминания `/teacher-login`/`AuthModal` ниже — устаревшие.
> **Верстка (2026-07-14):** страницы кабинета обёрнуты в `PageContainer` (центрируют контент, `mx-auto`, ширина `wide 1240px` / `form` / `narrow`) — контент больше не прижат влево. Списки (треки/словарь/заметки/материалы/мои тесты) отрисованы карточными **сетками** (2–3 колонки), detail-страница трека — **2 колонки** (роадмап + история). Формы/чтение — центрированы комфортной колонкой.

## Публичные страницы (без авторизации)

### `/` — Лендинг ✅

**Файл:** `src/pages/landing/LandingPage.jsx`

Тёмный технологичный стиль. Секции: Header (фиксированный, blur при скролле), Hero (анимированные слова + GSAP), Features (6 карточек), About, FAQ (аккордеон), Footer (CTA + «Я преподаватель» → `/teacher-login`).

Auth: открывает `AuthModal` — регистрация студента или вход.

---

### `/teacher-login` — Вход преподавателя ✅

**Файл:** `src/pages/auth/TeacherLoginPage.jsx`

Тёмная тема, анимированные точки canvas, форма входа. Проверка роли после входа (не teacher → ошибка). После успеха: полноэкранная анимация → редирект `/dashboard`.

---

### `/forgot-password` — Забыл пароль ✅

**Файл:** `src/pages/auth/ForgotPasswordPage.jsx`

Форма с email → `POST /auth/forgot-password` → всегда «проверьте почту» (защита от enumeration).

---

### `/reset-password` — Сброс пароля ✅

**Файл:** `src/pages/auth/ResetPasswordPage.jsx`

Новый пароль по токену из письма → `POST /auth/reset-password`.

---

### `/verify-email` — Подтверждение email ✅

**Файл:** `src/pages/auth/VerifyEmailPage.jsx`

Открывается по ссылке из письма (`?token=`). Показывает статус верификации.

## Защищённые страницы (AppLayout)

### `/welcome` — Приветствие ✅

**Файл:** `src/pages/welcome/WelcomePage.jsx`

Полноэкранный onboarding после регистрации/входа. GSAP анимация + 3 подсказки → «Перейти в кабинет».

---

### `/dashboard` — Кабинет ✅ (Variant C Premium)

**Файл:** `src/pages/dashboard/DashboardPage.jsx`

👨‍🏫 **Учитель:** KPI-плашки (студенты / уроки на неделе / долг учеников / посещаемость%), блок «Сегодня» (уроки дня с `lessonLink`), «Предстоящие», лента «Activity Feed», dropdown «+ Создать».

🎓 **Студент:** KPI-плашки (уроков на неделе / ДЗ к сдаче / посещаемость / долг), список ДЗ с дедлайнами + `lessonLink`-кнопка, лента последних оценок.

---

### `/calendar` — Календарь ✅

**Файл:** `src/pages/calendar/CalendarPage.jsx`

👤 FullCalendar daygrid с ru locale. Групповые (синие) + индивидуальные (розовые) уроки. Клик → модалка с деталями урока, `lessonLink`-кнопка, `chatLink`-кнопка.

---

### `/groups` — Список групп ✅

**Файл:** `src/pages/groups/GroupsPage.jsx`

👨‍🏫 Список карточек + кнопка «Создать группу» (модалка с расписанием, chatLink, pricePerLesson).

🎓 Блок «Приглашения в группы (N)» (C3) — входящие `Invitation{pending}` с «Принять»/«Отклонить»; после accept группа появляется в списке.

---

### `/groups/:id` — Детали группы ✅

**Файл:** `src/pages/groups/GroupDetailPage.jsx`

👨‍🏫 Три вкладки:
- **Студенты** — добавить реального / «+ Заглушка» (без аккаунта) / «+ Пригласить» по нику (InviteModal, C3) / «Перенести» заглушку на реального / убрать
- **Уроки** — список + генерация по расписанию + создание разового урока + просмотр урока
- **Настройки** — редактировать `name/schedule/pricePerLesson/chatLink/lessonLink`, удалить группу

`chatLink` → кнопка «💬 Чат группы» в шапке (обе роли) и в карточке урока.

---

### `/homework` — Домашние задания 🔶

**Файл:** `src/pages/homework/HomeworkPage.jsx`

👨‍🏫 Создание ДЗ (для группового или инд. урока) + QuizGenerator (AI-генерация тестов через Groq), просмотр списка, удаление, `SubmissionsModal` (показывает имя ученика + файл + коммент), выставление оценки.

🎓 Список ДЗ со статусами (Не сдано / Просрочено / На проверке / Оценено) + `StudentHWCard`: сдача файлом (Cloudinary, необязательно) + комментарий.

**Нет:** ownership check на edit/delete на бэке (⚠️ backlog)

---

### `/attendance` — Посещаемость ✅

**Файл:** `src/pages/attendance/AttendancePage.jsx`

👤 **3 вкладки (dual-confirmation):**

- **Журнал** — сетка ученики×даты; учитель отмечает/редактирует; студент видит свою историю
- **Ожидают (N)** — студент подтверждает или оспаривает каждую запись учителя
- **Спорные (N)** — учитель разрешает споры: принять или отклонить претензию студента

---

### `/payments` — Оплаты ✅

**Файл:** `src/pages/payments/PaymentsPage.jsx`

👨‍🏫 Таблица долгов по ученикам (начислено − оплачено = долг); кнопка «Внести оплату» → модалка (сумма + метод) + `ConfirmDialog`; история оплат с фильтрами; сводка по методам.

🎓 Долг по каждому учителю + кнопка «Оплатить» → `/pay/:teacherId`.

---

### `/pay/:teacherId` — Оплата учителю ✅ (2026-07-09)

**Файл:** `src/pages/pay/PayPage.jsx`

🎓 Реквизиты учителя (IBAN/BLIK/PayPal/Revolut/custom) + загрузка скриншота перевода → `POST /payments/record { source:'student' }`. Авто-запись `PaymentRecord`.

### `/students` — Студенты ✅

👨‍🏫 Таблица студентов с поиском по имени. Заглушки помечены (нет аккаунта). Кнопка «Перенести» → merge на реального ученика.

### `/individual-courses` — Инд. курсы ✅

**Файл:** `src/pages/individual-courses/IndividualCoursesPage.jsx`

👨‍🏫 Список контрактов (студент + расписание + цена) + создание/редактирование/удаление + «Сгенерировать уроки».

---

### `/individual-lessons` — Инд. уроки 🔶 (read-only)

**Файл:** `src/pages/individual-lessons/IndividualLessonsPage.jsx`

👤 Список уроков с фильтром по периоду. Данные используются также в Calendar.

**Нет:** создание/редактирование уроков через эту страницу.

---

### `/profile` — Профиль ✅

**Файл:** `src/pages/profile/ProfilePage.jsx`

👤 Instagram-style: обложка + аватар (upload Cloudinary), bio, соцсети, языки, прогресс-бар заполнения.

**Табы:**
- **Профиль** — просмотр личных данных
- **Аналитика** — Recharts (учитель: Line paid+charged + Area студенты + плашка attendance; студент: Area посещаемость + Line оценки + ДЗ progress-bar)
- **Безопасность** — смена пароля

---

### `/settings` — Настройки ✅ (2026-07-09)

**Файл:** `src/pages/settings/SettingsPage.jsx`

👤 Замена «редактировать профиль» — 2 вкладки:
- **Личные данные** — имя, username, bio, аватар, языки, соцсети (без обложки)
- **Способы оплаты** (только учитель) — IBAN / BLIK / PayPal / Revolut / custom; сохраняет в `paymentDetails JSONB`

Баннер «Заполните реквизиты» и «!» в сайдбаре пока поля пусты.

---

### `/help` — Справка ✅

**Файл:** `src/pages/help/HelpPage.jsx`

👤 FAQ-секции: группы, уроки, ДЗ, посещаемость, финансы, видеозвонки (lessonLink), тарифы, PWA установка.

---

### `/admin` — Администрирование ✅ (2026-07-09)

**Файл:** `src/pages/admin/AdminPage.jsx`

👑 Доступна только `role=admin` через `RoleRoute`.

- **Stats** — KPI: учителя / студенты / группы / уроки / выручка
- **Teachers** — пагинированный список с тарифом и статусом active/emailVerified
- **Users** — таблица всех пользователей (фильтр role/active); dropdown «Действия ▾» на каждой строке (createPortal — не уходит под таблицу):
  - Смена роли (teacher/student/admin)
  - Смена тарифа (free/pro/school)
  - Деактивировать / Восстановить доступ
